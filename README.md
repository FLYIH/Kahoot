# README

## Project Name
Network Programming Final Project - Kahoot

## Steps to Start

### 1. Place the Project under `unpv13e`
Move the `Kahoot` folder into the root directory of `unpv13e`.

### 2. Compile the Project
Navigate to the `Kahoot` folder and run:

```bash
cd unpv13e/Kahoot
make
```

### 3. Run the Programs

#### Run the Server
```bash
./server
```

#### Run the Client (GUI)
```bash
./client_gui
```
